## PhpStorm settings

### 使用方法

1. 打开PhpStorm
2. Preferences -> Tools -> Settings Repository [首设置->工具->设置存储库]
3. Read-only Sources 点击左下角加号，输入本项目地址（最好使用https地址）
```https://github.com/HogenYuan/codestyles.git```
4. 点击两次OK保存
5. 完全退出PhpStorm并重新打开
6. Perferences -> Editor -> Code Style [首设置->编辑器->code style]
7. Scheme 更改为 Hogen
8. 点击OK保存
9. 完成
